使用blogdown迁移hugo博客

2020.11.10

对 blogdown 还是念念不忘，因此，今天将原来的博客重新使用 blogdown 创建了一下，下面的内容是记录过程摘要：

建站

library(blogdown)
new_site(theme = "varkai/hugo-theme-zozo")
install_hugo(forece = TRUE)

将原来的hugo博客文件夹下的内容，覆盖新的 blogdown 之下的内容，当然是要覆盖，不保留新创建的内容，这样就不用再配置了，还是原来的配置。

使用下面的命令创建本博文：

blogdown:::new_post_addin()
serve_site()

gitpages

删除原来的 rep，重新创建 zhujiedong.github.io 的 repo，使用 git 将 public 目录上传：

以下代码使用 git bash

cd g：
cd my-blog/public
git init
git add .
git commit -m "first commit"
git branch -M main
#git remote add origin https://github.com/zhujiedong/zhujiedong.github.io
# do not need, already exists after copy, if not need exexcute
git push -u origin main

评论系统

(valine)[https://valine.js.org/] 是国产的评论系统，在 (leancloud)[https://leancloud.cn/] 注册登陆获取appid，然后在 toml 文件设置好 appid 即可。
